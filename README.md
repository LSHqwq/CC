# 图像识别分类器

一个功能强大的智能奖状分类工具，支持多种文件格式，具有高识别精度和智能分类能力。

## 🚀 主要功能

### 核心功能
- **多种文件格式支持**：PDF、JPG、PNG、BMP、TIFF、JPEG
- **智能文本识别**：
  - PDF文件：直接提取文本，精度极高
  - 图片文件：使用Tesseract OCR识别，清晰扫描件精度达95%以上
  - 奖状文档：格式规范的奖状识别精度可达98%以上
- **智能分类条件**：支持按姓名、年份、奖项类型、颁发机构、奖项级别等多种条件分类
- **模糊输入理解**：支持自然语言输入，模糊匹配分类意图
- **批量高效处理**：一键批量分类大量文件
- **美观现代化界面**：卡片式设计，渐变色彩，直观易用

### 技术特点
- **本地运行**：无需网络连接，数据安全
- **独立可执行文件**：无需安装Python环境
- **自动预处理**：图片自动灰度转换、二值化提高识别效果
- **智能文件夹命名**：根据分类条件自动生成有意义的文件夹名称

## 📦 安装说明

### 方法一：使用安装包（推荐）
1. 下载 `图像识别分类器.zip` 压缩包
2. 解压到任意位置
3. 运行 `安装程序.bat`
4. 按照提示安装Tesseract OCR（首次使用必需）
5. 双击桌面"开始分类"快捷方式启动

### 方法二：从源码运行
1. 确保已安装Python 3.7或更高版本
2. 克隆本项目到本地
3. 安装依赖包：
   ```
   pip install -r requirements.txt
   ```
4. 安装Tesseract OCR并配置环境变量
5. 运行：
   ```
   python app.py
   ```

## 🛠️ 使用方法

1. **选择源文件夹**：点击"浏览"按钮选择包含奖状的文件夹
2. **选择目标文件夹**：点击"浏览"按钮选择分类结果保存位置
3. **设置分类条件**：在输入框中输入分类条件（如"姓名"、"年份"、"省级"等）
4. **开始分类**：点击"开始分类"按钮
5. **查看结果**：在目标文件夹中查看分类好的文件

## 🔧 技术实现

- **界面**：tkinter + 现代化设计
- **PDF处理**：PyPDF2、PyMuPDF
- **图像识别**：Tesseract OCR、OpenCV、Pillow
- **文本分析**：正则表达式、模糊匹配算法
- **打包工具**：PyInstaller
- **网站**：HTML5 + CSS3 + GitHub Pages

## ⚠️ 注意事项

1. **Tesseract OCR**：首次使用需要安装Tesseract OCR引擎并选择安装中文语言包
2. **图片质量**：识别精度取决于图片质量，建议使用清晰的扫描件或高质量照片
3. **性能**：处理大量文件时可能需要较长时间，请耐心等待
4. **兼容性**：支持Windows 10/11系统

## 🎯 识别精度说明

- **PDF文件**：直接提取文本，精度极高
- **图片文件**：
  - 清晰扫描件：95%以上识别精度
  - 高质量照片：90%以上识别精度
  - 模糊图片：识别精度会下降
- **奖状文档**：格式规范的奖状识别精度可达98%以上

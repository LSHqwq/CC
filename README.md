# 奖状分类器

一个简单易用的奖状分类工具，可以根据不同条件自动分类PDF格式的奖状。

## 功能特点

- 支持按姓名、奖项类型、年份、颁发机构等多种方式分类
- 自动从PDF中提取信息
- 美观的图形界面
- 操作简单易懂
- 本地运行，无需网络连接

## 安装说明

1. 确保已安装Python 3.7或更高版本
2. 克隆或下载本项目到本地
3. 安装依赖包：
   ```
   pip install -r requirements.txt
   ```

## 使用方法

1. 运行软件：
   ```
   python app.py
   ```

2. 在界面中：
   - 点击"浏览"按钮选择奖状源文件夹（包含PDF格式的奖状）
   - 点击"浏览"按钮选择分类目标文件夹（分类后的奖状将存放在此）
   - 选择分类方式（按姓名、奖项类型、年份或颁发机构）
   - 点击"开始分类"按钮开始分类
   - 分类完成后会显示完成提示

## 技术实现

- 使用tkinter构建图形界面
- 使用PyPDF2和PyMuPDF提取PDF文本
- 使用正则表达式从文本中提取信息
- 根据提取的信息创建相应的文件夹并分类文件

## 注意事项

- 目前的信息提取规则较为简单，可能需要根据实际奖状格式进行调整
- 请确保PDF文件可正常打开和提取文本
- 分类过程中请不要关闭软件窗口

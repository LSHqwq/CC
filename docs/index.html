使用方法

  1. 下载并解压 图像识别分类器.zip 文件
  2. 双击 图像识别分类器.exe 运行程序
  3. 点击"浏览"按钮选择包含奖状的源文件夹
  4. 点击"浏览"按钮选择分类后保存的目标文件夹
  5. 在"分类条件"输入框中输入分类方式（如"姓名"、"年份"、"省级"等）
  6. 点击"开始分类"按钮开始自动分类
  7. 分类完成后，奖状会按指定条件自动分类到相应的文件夹中
